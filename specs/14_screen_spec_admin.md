# 14 — Screen Spec Admin

## Ziel

Die Admin-Seite bleibt ruhig und eingeklappt. Sie dient in der Referenzwelt vor allem zur Sicht auf:

- User
- Gruppen
- Memberships
- Referenzstruktur

## Führender Referenzstand

User:

- Admin
- Service Auftrag / Freigabe
- Service Durchfuehrung / Dokumentation
- Produktion Auftrag / Freigabe
- Produktion Durchfuehrung / Dokumentation

Gruppen:

- Kundenservice
- Produktion

## UI-Regel

- Karten sind standardmäßig collapsed
- keine lauten Erklärungstexte
- Namen und Rollen sind fachlich lesbar
